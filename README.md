# NodeChat

A small chat system made with node.js, express, and socket.io.

Supports:
* Usernames
* Changing Usernames
* Connection and Disconnection messages
* Dynamic User List
* Private messaging (/w [username] [message])
* Extendable Commands (of the form /* [cmd] [options])
